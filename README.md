ISTRUZIONI PER L'USO DEGLI SCRIPT:

1) Scaricare i file tracciatore.py e simulazione.py nella stessa cartella

2) Utilizzare emacs per aprire il file tracciatore.py e raggiungere l'ultima
   riga di codice (riga 132). Sostituire l'argomento del comando
   sys.path.append() con il percorso della cartella nella quale sono stati
   salvati gli script python

3) Il codice simulazione.py usa il modulo argparse, dunque per avere
   informazioni più dettagliate sulle funzionalità dello script è sufficiente,
   dal terminale, eseguire il programma utilizzando il comando -h oppure --help
   in modo analogo a quanto scritto sotto

   python3 simulazione.py -h

   oppure

   python3 simulazione.py --help

   Eseguire lo script senza utilizzare nessuno degli argomenti elencati dai
   comandi sopracitati non produce nessun risultato visibile.
